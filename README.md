### **→ [Open a new issue](https://github.com/Serilum/.issue-tracker/issues/new/choose)**

### Please keep a few things in mind:

- **Tell me the mod name.**
  - This issue tracker is used for all of Serilum's mods. To avoid confusion, please specify the name.
- **Have a quick look at the <a href="https://github.com/Serilum/.information/wiki/FAQ" target="_blank">FAQ</a> first.**
  - Common questions about supported versions, timeframes, downloading and mod configuration may already be answered there.
- **Make sure you are on the latest mod version and a support Minecraft version.**
  - The issue you ran into may already be fixed.
